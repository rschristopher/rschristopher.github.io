    Level 5: Airgap Quarantine¶

    Save Bitcoin with greater security than any bank (as much as you'd be comfortable hiding under your mattress).

    Level 5 implements an airgap quarantine, ensuring your signing device and its associated computer never connect to any network or external device, maintaining complete isolation. By using QR codes for all data transfers and a dedicated air-gapped computer for transaction signing, you eliminate risks from physical connections (e.g., USB, microSD) and network exposure. This setup builds on Level 4, combining a Bitcoin Knots full node, a Tails OS transaction manager, and a Coldcard with QR display functionality. Alternatives like SeedSigner and Specter-DIY are also discussed.

    No paid nor influenced content -- all views are from personal experience

    Why Airgap Quarantine?¶

    An airgap quarantine maximizes security by isolating the signing device and its computer, preventing exposure to malware or network attacks. Benefits include:

    • Maximum Security: No network or physical connections eliminate key compromise vectors.
    • Privacy: QR codes transfer data optically, preventing interception via USB or network.
    • Sovereignty: Full control over your Bitcoin with verifiable, open-source tools.

    This guide uses a Coldcard Mk4 (with QR code support) and a dedicated air-gapped computer, ensuring a strict quarantine. Alternatives like SeedSigner and Specter-DIY offer similar QR-based workflows.

    Setting Up Your Airgap Quarantine¶

    Level 5 assumes you have a Level 4 setup: a Bitcoin Knots full node and a Tails OS transaction manager with Electrum or Sparrow for a watch-only wallet. The key changes are using a Coldcard with QR display, a separate air-gapped computer for signing, and QR codes for all data transfers.

    Coldcard Setup¶

    The Coldcard Mk4 supports QR code display for xPub export and transaction signing, ideal for airgap quarantine. You’ll need:

    • Hardware: Coldcard Mk4, 9V battery or offline power source, microSD card (for initial setup only).
    • Software: Latest Coldcard firmware from coldcard.com.

    • Environment: A clean, offline computer (e.g., temporary Tails OS session) for initial setup.

    • Initialize Coldcard:

    • Power on the Coldcard using a 9V battery or offline charger (never connect to a networked device).
    • Update the firmware offline using a microSD card, following Coldcard’s upgrade guide.
    • Create a new 12/24-word seed phrase (BIP-39) or import an existing seed.
    • Write down the seed phrase on paper or metal and store it securely offline.
    • Export xPub via QR:
    • From the Coldcard menu, select Advanced/Tools > Export Wallet > BIP-84 (or desired derivation path).
    • Choose QR Code to display the xPub.
    • On your Tails OS transaction manager, open Electrum or Sparrow, create a watch-only wallet, and scan the xPub QR code (similar to Level 3).
    • Verify the watch-only wallet shows correct addresses and balances via your Bitcoin Knots node.
    • Secure Coldcard:
    • Set a strong PIN and enable duress PIN for added security.
    • Store the Coldcard in a tamper-evident bag or safe when not in use.

    Air-Gapped Computer Setup¶

    A dedicated air-gapped computer processes transactions offline, interfacing with the Coldcard via QR codes. This ensures no network exposure during signing.

    • Hardware: An old laptop or Raspberry Pi with a display, webcam, and no network hardware (e.g., remove Wi-Fi/Bluetooth modules or use a model without them).
    • Storage: A USB drive with Tails OS or a clean Linux install (e.g., Ubuntu).
    • Software: Install Sparrow or Electrum offline for transaction processing.

    • QR Scanner: Ensure the computer has a webcam or attach a USB webcam compatible with your OS.

    • Prepare the Computer:

    • Physically disable network capabilities (e.g., remove Wi-Fi card, tape over Ethernet port).
    • Install Tails OS or Ubuntu on a USB drive using a clean, offline system.
    • Install Sparrow or Electrum offline by transferring the verified installer via a USB drive.
    • Verify Integrity:
    • Boot the air-gapped computer and confirm no network connections are possible (e.g., check ifconfig or nmcli for no active interfaces).
    • Use a fresh Tails OS session or a dedicated Linux install for each signing session to minimize risks.

    Transaction Manager Setup¶

    The transaction manager remains the Level 4 setup: a Tails OS USB with Electrum or Sparrow, connected to your Bitcoin Knots full node. Ensure it has a webcam for scanning QR codes from the air-gapped computer or Coldcard.

    • Electrum: Enable QR code support in Preferences > QR Code.
    • Sparrow: Use Tools > Scan QR Code for QR-based workflows.
    • Privacy: Run on a local network with your Knots node, using Tails OS’s Tor routing for broadcasting transactions.

    Airgap Transaction Workflow¶

    The airgap quarantine uses QR codes to transfer data between the transaction manager, air-gapped computer, and Coldcard, ensuring no physical or network connections.

    Step 1: Create Unsigned Transaction¶

    1. On the Tails OS transaction manager, open Electrum or Sparrow.
    2. Create a transaction in your watch-only wallet (e.g., send Bitcoin to an address).
    3. Export the unsigned transaction as a QR code:
    4. In Electrum, select Send > QR Code.
    5. In Sparrow, go to Transactions > Export > QR Code.
    6. Verify the transaction details (amount, address, fee) on the transaction manager.

    Step 2: Transfer to Air-Gapped Computer¶

    1. Boot the air-gapped computer with Tails OS or your Linux install.
    2. Open Sparrow or Electrum and scan the QR code from the transaction manager’s screen using the webcam.
    3. For large transactions, scan multiple QR codes as prompted.
    4. Verify the transaction details (address, amount, fee) on the air-gapped computer.
    5. Save the unsigned transaction and export it as a QR code for the Coldcard.

    Step 3: Sign with Coldcard¶

    1. Power on the Coldcard using a battery or offline charger.
    2. Insert a microSD card (optional, for backup) or rely solely on QR codes.
    3. Select Ready to Sign > QR Code from the menu.
    4. Scan the unsigned transaction QR code from the air-gapped computer’s screen using the Coldcard’s camera.
    5. For large transactions, scan multiple QR codes as prompted.
    6. The Coldcard displays transaction details (destination address, amount, fee). Double-check these against the air-gapped computer’s details.
    7. If correct, press OK (✓) to sign. The Coldcard generates a QR code for the signed transaction.
    8. If incorrect, press X to reject and investigate discrepancies.

    Step 4: Transfer Signed Transaction¶

    1. On the air-gapped computer, scan the signed transaction QR code from the Coldcard’s screen.
    2. Verify the signed transaction details in Sparrow or Electrum.
    3. Export the signed transaction as a QR code.

    Step 5: Broadcast Signed Transaction¶

    1. On the transaction manager, scan the signed transaction QR code from the air-gapped computer’s screen:
    2. In Electrum, use Tools > Load Transaction > From QR Code.
    3. In Sparrow, use Tools > Scan QR Code.
    4. Verify the transaction details again (address, amount, fee).
    5. Broadcast the transaction to the Bitcoin network via your Bitcoin Knots node.
    6. Confirm the transaction appears in your Knots node’s mempool (e.g., via bitcoin-cli getmempoolentry).

    Security Notes¶

    • Screen Distance: Position devices 6–12 inches apart for clear QR code scanning.
    • Lighting: Use a well-lit environment, avoiding direct sunlight on screens.
    • Verification: Cross-check transaction details on all devices (transaction manager, air-gapped computer, Coldcard) to prevent spoofing.
    • Power Off: Turn off the Coldcard and air-gapped computer after signing, storing them securely.

    Maintaining the Airgap¶

    To preserve the strict airgap quarantine: - No Connections: Never connect the Coldcard or air-gapped computer to the internet, USB ports (except for offline power), or networked devices. - Offline Power: Use a 9V battery for the Coldcard and a power bank or offline charger for the air-gapped computer. - Secure Storage: Store the Coldcard and air-gapped computer in tamper-evident bags or a safe when not in use. - Clean Setup: Update Coldcard firmware or air-gapped computer software using offline Tails OS sessions, then destroy the session. - Backup Integrity: Store seed phrase backups (paper/metal) in separate, secure locations, never digitizing them. - Monitor Transaction Manager: Periodically use a fresh Tails OS USB for the transaction manager to minimize malware risks.

    Comparing QR-Capable Devices¶

    While Coldcard is used here, other QR-capable signing devices offer similar airgap quarantine capabilities:

    • Coldcard Mk4: Bitcoin-only, air-gapped, with QR code support for xPub export and transaction signing. Its robust security (secure element, PIN protection) and compact design make it ideal for strict quarantine, though it’s pricier (~$150). coldcard.com.
    • SeedSigner: A DIY, open-source signing device built on Raspberry Pi Zero. Fully transparent, low-cost (~$50–$100), and QR-native, but requires assembly and technical expertise. Ideal for those prioritizing open hardware. seedsigner.com.
    • Specter-DIY: An open-source, DIY signing device using affordable hardware (e.g., ESP32). Supports QR codes and air-gapped workflows but requires significant setup and technical skills. Best for advanced users. github.com/cryptoadvance/specter-diy.

    Coldcard’s ease of use and security features make it a strong choice, while SeedSigner and Specter-DIY appeal to those prioritizing cost and open hardware.

    Conclusion¶

    Level 5 airgap quarantine with a Coldcard, dedicated air-gapped computer, and QR codes offers unmatched security for your Bitcoin, surpassing traditional banking systems. By integrating a Bitcoin Knots full node, Tails OS transaction manager, and a strictly air-gapped signing process, you achieve ultimate sovereignty and privacy. This setup is a digital fortress hidden under your mattress, accessible only by you.

    To get started: 1. Set up a Coldcard with an offline seed and export its xPub via QR code. 2. Configure a dedicated air-gapped computer with Sparrow or Electrum. 3. Ensure your Tails OS transaction manager has a webcam for QR scanning. 4. Practice the QR code transaction workflow with small amounts to build confidence. 5. Store the Coldcard, air-gapped computer, seed phrase backups, and node configuration securely offline.

    For further reading, explore the Coldcard documentation, SeedSigner guides, Specter-DIY resources, or Bitcoin Knots guides.
